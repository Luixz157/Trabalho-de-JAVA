1. Registrar Usuário
Descrição: Permite que novos usuários se registrem no sistema para poderem fazer doações. Ator: Usuário Fluxo Principal:
O usuário acessa a página de registro.
O usuário fornece informações pessoais (nome, e-mail, telefone, etc.).
O sistema valida e armazena as informações.
O sistema confirma o registro e permite o login.

2. Fazer Login
Descrição: Permite que usuários registrados façam login no sistema. Ator: Usuário Fluxo Principal:
O usuário acessa a página de login.
O usuário insere suas credenciais (e-mail e senha).
O sistema valida as credenciais.
O sistema concede acesso ao painel de usuário.

3. Agendar Doação
Descrição: Permite que usuários agendem uma doação, especificando itens, local e data. Ator: Usuário Fluxo Principal:
O usuário faz login no sistema.
O usuário acessa a seção de agendamento de doação.
O usuário preenche os detalhes da doação (itens, local, data).
O sistema confirma o agendamento e fornece um resumo.

4. Agendar Retirada de Doação
Descrição: Permite que os usuários agendem a retirada das doações. Ator: Usuário Fluxo Principal:
O usuário faz login no sistema.
O usuário acessa a seção de agendamento de retirada.
O usuário escolhe a data e o horário para a retirada.
O sistema confirma o agendamento e fornece um resumo.

5. Confirmar Doação
Descrição: Permite que o sistema confirme o recebimento das doações e atualize o status. Ator: ONG Fluxo Principal:
A ONG acessa o painel de administração.
A ONG visualiza as doações agendadas.
A ONG confirma o recebimento das doações.
O sistema atualiza o status da doação.

6. Gerar Relatórios
Descrição: Permite que a ONG gere relatórios sobre as doações recebidas. Ator: ONG Fluxo Principal:
A ONG acessa a seção de relatórios.
A ONG seleciona o tipo de relatório desejado (por data, tipo de item, etc.).
O sistema gera e exibe o relatório.
A ONG pode exportar o relatório para diferentes formatos (PDF, Excel, etc.).

7. Atualizar Dados de Perfil
Descrição: Permite que os usuários atualizem suas informações de perfil. Ator: Usuário Fluxo Principal:
O usuário faz login no sistema.
O usuário acessa a seção de perfil.
O usuário atualiza suas informações.
O sistema salva as alterações e confirma a atualização.

8. Cancelar Agendamento de Doação
Descrição: Permite que os usuários cancelem um agendamento de doação. Ator: Usuário Fluxo Principal:
O usuário faz login no sistema.
O usuário acessa a lista de agendamentos.
O usuário seleciona o agendamento a ser cancelado.
O sistema confirma o cancelamento e atualiza o status.

9. Notificar Sobre Doação
Descrição: Permite que o sistema envie notificações por e-mail ou SMS para lembrar sobre o agendamento de doação. Ator: Sistema Fluxo Principal:
O sistema verifica os agendamentos próximos.
O sistema envia notificações de lembrete para os usuários.
O sistema atualiza o status de notificação.
